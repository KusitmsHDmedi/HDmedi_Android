## Tech Stack
| Name              | Description                                               |
| ----------------- | --------------------------------------------------------- |
| Retrofit          | HTTP 통신을 도와주는 라이브러리                           |
| LiveData          | 생명주기 상태에 따라 UI 업데이트 등의 작업을 자동으로 관리하는 라이브러리 |
| ViewModel         | 수명주기를 고려하여 데이터를 저장하고 관리하는 라이브러리 |
| Coroutine         | 비동기 처리 라이브러리                                    |
| DataBinding        | 프래그먼트를 쉽게 관리할 수 있도록 도와주는 라이브러리    |
| NaverLoginAPI      | 네이버 소셜 로그인을 구현하기 위한 라이브러리 |
